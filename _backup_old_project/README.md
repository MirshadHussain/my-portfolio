# 3D Portfolio Website

Welcome to the 3D Portfolio repository. This project is a full-stack application built following real-world software engineering practices.

## Repository Structure

- **`frontend/`**: The client-side web application (UI, 3D scenes, state management).
- **`backend/`**: The server-side logic, business rules, and API endpoints.
- **`database/`**: Database setup scripts, Docker configurations, schema definitions, and migrations.
- **`docs/`**: Project documentation, architectural diagrams, API contracts, and setup instructions.
